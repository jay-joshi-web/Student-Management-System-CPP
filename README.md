# Student Management System (C++)

A menu-driven Student Management System developed in C++ using Object-Oriented Programming (OOP) and File Handling concepts.

## Features

- Add Student Records
- Display All Student Records
- Search Student by Roll Number
- Delete Student Records
- Persistent Data Storage using Text Files
- Interactive Menu-Based Interface

## Technologies Used

- C++
- Object-Oriented Programming (Classes & Objects)
- File Handling (ifstream, ofstream)
- Functions
- Git & GitHub

## Project Structure

text main.cpp      -> Source Code 
student.txt   -> Stores Student Records 
README.md     -> Project Documentation 

## How It Works

Each student record contains:

- Name
- Roll Number
- Marks

Records are stored in a text file and can be accessed, searched, displayed, and deleted through the menu-driven interface.
